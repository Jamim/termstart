# Usage

Keys:
- `Enter and Space`: focus prompt

Commands:
- `ls` - list all sites
- `open <site's name>` open a site
- `search <something>` search with duckduckgo
- `del <site's name>` delete a site
- `add <site's name> <site's url>` add a site
